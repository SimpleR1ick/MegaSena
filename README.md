# MegaSena

Projeto desemvolvido com java, simula um sorteio da Mega Sena.

Funções do sistema.
1 - Cadastrar um Jogador
2 - Cadastrar uma aposta
3 - Iniciar o sorteio e listas os vencedores
4 - Sair

1) O usuario digita os dados do apostador do quais são salvos em um arquivo.txt
2) O sistema lista todos os apostadores cadastrados, o usuario entra com a quantidade de jogadores na aposta, sendo o primeiro o organizador, seguindo a quantidade de numeros apostados, minimo 6 e maximo 15.
3) E inserido os 6 numeros premiados é o sistema verifica se alguma das apostas foi vencedora, caso sim, 10% e reservado de bonus para o organizador da aposta e o resto e redividido com todos os apostadores
4) Por fim, e possivel encerrar o sistema na opção sair
